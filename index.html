一、什么是C语言？

C语言是一种通用的、过程式的计算机编程语言，广泛应用于系统级编程，如操作系统、编译器、数据库等。它的设计提供了低级处理的功能，包括内存管理、数据类型定义、以及直接与硬件通信等。由于其高效、灵活和可移植性，C语言至今仍被广泛应用于各种领域。

二、编写“Hello World”程序

在编程的世界里，每个初学者都会从“Hello World”这个程序开始。这是一个非常简单的程序，它仅仅在屏幕上打印出“Hello, World!”这句话。但是，它标志着你的编程之旅已经闪亮启航！ 

  
#include <stdio.h>  
  
int main() {  
    printf("Hello, World!\n");  
    return 0;  
}

让我们来逐行解释这个程序：

  • #include <stdio.h>：这是一个预处理指令，它告诉编译器在实际编译之前要包含标准输入输出头文件（stdio.h）。
  • int main()：这是程序的主函数，程序的执行从这里开始。
  • printf("Hello, World!\n")：这是C语言中的标准输出函数，用于在屏幕上打印字符串。

三、编译和运行程序

编写完程序后，我们需要使用编译器将源代码编译成机器语言，然后才能运行。在大多数操作系统中，我们可以使用GCC（GNU Compiler Collection）编译器来编译C语言程序。

假设你的源代码文件名为hello.c，你可以在命令行中输入以下命令来编译它： gcc hello.c -o hello 编译成功后，你就可以在命令行中输入以下命令来运行程序： ./hello
